Amber Router

Written: 2017-2018

Abstract: The Amber Router is a project contribution to the open source Amber Framework, which is a rails-like web framework in Crystal.

The original routset library for Amber was based on Radix::Tree, which was abandoned and incomplete enough to warrant a rewrite for speed and predictability. The rewrite was advocated and initiated by me, and ongoing open source contributions by a few community members.

Special consideration was given to runtime routing performance, visualizing the route tree for developers, and API compatibility with the existing Radix::Tree implementation.
