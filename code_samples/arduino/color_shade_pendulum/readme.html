Color Shade Pendulum

Written: August, 2012

Abstract: An artist contacted me one day to ask about programming an arduino for a kinetic sculpture made out of a series of pendulums of varying lengths designed to simulate a murmuration of starlings, to be installed at Burning Man 2012.

Compiling Status: unknown
